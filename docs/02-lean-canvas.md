| **1. Problema** | Top 3 dores: |
| | • Jovens 18-30 sentem ansiedade frequente mas não conseguem mapear seus padrões emocionais ao longo do tempo. |
| | • Apps de meditação genéricos (Headspace, Lojong) não personalizam e geram baixo engajamento. |
| | • Terapia digital (Cíngulo, Zenklub) é financeiramente inacessível para o público em início de carreira (R$30-200/mês). |
| | **Alternativas existentes:** anotar no Notes do celular, conversar com amigos, scroll infinito como escape, abandonar apps de meditação. |
| **2. Segmentos de Clientes** | • **Primário:** jovens 18-30 anos no Brasil, universitários ou em início de carreira, com sintomas frequentes de ansiedade, renda R$1.500-R$5.000/mês, fluentes em uso de IA conversacional (já usam ChatGPT). |
| | • **Early adopters:** estudantes universitários que já tentaram apps de meditação e abandonaram, ou que já usam ChatGPT para "desabafar". |
| **3. Proposta de Valor Única** | "Entenda seu emocional como você entende sua agenda. Um diário com IA que escuta, identifica padrões e devolve clareza — em português, privado e por menos de R$1/dia." |
| | **Conceito de alto nível:** "O Anki para a sua mente." |
| **4. Solução** | • Diário em texto livre com classificação automática de emoções por IA. |
| | • Resumo semanal personalizado com 1 reflexão acionável. |
| | • Linha do tempo emocional visual (4 semanas). |
| | • Política de privacidade transparente (dados não treinam modelos públicos). |
| **5. Canais** | • Orgânico: Instagram (Reels educativos sobre saúde mental), TikTok, parcerias com criadores de conteúdo de bem-estar. |
| | • Comunidades: grupos de WhatsApp/Discord de universidades, subreddits brasileiros (r/Brasil, r/desabafos). |
| | • Indicação: programa "indique 1 amigo, ganhe 1 mês". |
| | • SEO de cauda longa: "como saber se tenho ansiedade", "diário emocional online". |
| **6. Fontes de Receita** | • **Freemium**: 7 dias grátis + 3 entradas/semana grátis para sempre. |
| | • **Plano Pro**: R$19,90/mês ou R$159/ano (R$13/mês) — entradas ilimitadas, resumos semanais, linha do tempo completa. |
| | • **B2B futuro (fora do MVP)**: licenças para universidades, RHs de empresas, planos de saúde. |
| **7. Estrutura de Custos** | • Custo de inferência da IA (LLM): ~R$0,30 por usuário ativo/mês. |
| | • Infraestrutura (cloud, banco de dados criptografado): ~R$0,50 por usuário/mês. |
| | • Marketing orgânico: tempo do fundador (custo de oportunidade). |
| | • Compliance LGPD + auditoria de privacidade: R$8.000 inicial + R$1.500/mês. |
| **8. Métricas-Chave** | • **Ativação**: % de usuários que fazem ≥ 3 entradas na 1ª semana. |
| | • **Retenção D30**: % de usuários ativos 30 dias após cadastro. |
| | • **Conversão freemium→pago**: % que assina após período grátis. |
| | • **NPS**: medido a cada 30 dias de uso. |
| | • **Tempo médio entre entradas** (proxy de hábito formado). |
| **9. Vantagem Injusta** | • **Domínio do PT-BR**: prompts e tom calibrados culturalmente para brasileiros (vs Wysa em inglês). |
| | • **Posicionamento ético**: privacidade radical é diferencial defensável vs Big Tech. |
| | • **Análise temporal**: ChatGPT esquece a conversa anterior; CalmAI lembra e correlaciona. |
| | • **Comunidade de cocriação**: usuários iniciais ajudam a calibrar tom (rede de confiança difícil de copiar). |

## Resumo executivo do canvas

CalmAI ataca um mercado de **R$ multi-bilhões em saúde mental digital** com um posicionamento único: **diário emocional com IA, em português, com privacidade radical, por menos de R$20/mês**. O modelo freemium reduz fricção de aquisição, e o custo de inferência baixo (~R$0,80/usuário/mês) permite margem saudável mesmo no plano básico. A vantagem injusta combinada (PT-BR + ética + temporalidade + comunidade) cria barreira de entrada difícil de copiar para concorrentes globais e nacionais.
